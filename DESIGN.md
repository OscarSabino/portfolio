Vibrant Minimalism - Portfolio Design System

Visual Identity

A high-contrast, developer-centric aesthetic that balances brutalist precision with vibrant digital accents. The system uses a deep dark mode foundation to allow electric accent colors and bold typography to command attention.

Design Tokens

Colors





Surface (Background): #131314 (Deep Charcoal) - Provides a stable, professional base.



Primary (Accent): #00f0ff (Electric Cyan) - Used for primary actions, success states, and key highlights.



Secondary (Accent): #bf5af2 (Neon Purple) - Used for tags, secondary highlights, and code elements.



Surface Bright: #3a393a - Used for card borders and elevated surfaces.



Text Primary: #ffffff (Pure White) - For maximum legibility on dark backgrounds.



Text Secondary: #9ca3af (Gray 400) - For metadata, descriptions, and supporting text.

Typography





Primary Font: Space Grotesk - A geometric sans-serif with a technical, modern feel.



Headlines: Large, bold (font-bold), and tight tracking (-0.05em). Often uppercase for a brutalist impact.



Body: Clean, legible weight (font-normal) with generous line-height for readability.



Mono: JetBrains Mono or similar system monospaced fonts for code snippets and "terminal" accents.

Components & Patterns





Cards: Subtle borders (1px solid #3a393a), deep backgrounds, and rounded corners (ROUND_EIGHT).



Buttons: High-contrast backgrounds (Primary Cyan) with dark text for clear calls to action.



Navigation: Floating, blurred backgrounds (glassmorphism) to maintain context while scrolling.



Spacing: Tight, grid-based spacing system using a 4px or 8px base unit to ensure alignment and rhythm.

Design Principles





Precision Over Fluff: Every element should feel intentional. Avoid unnecessary shadows or gradients.



High Information Density: Clear hierarchy through font size and weight rather than complex layouts.



Developer Soul: Integrate code-like elements (terminal prompts, mono fonts, syntax highlighting colors) as decorative but meaningful accents.
